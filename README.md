# Mayigushi-App

## 概要

该项目是蚂蚁故事网开发的app，其下包含2个小项目，公用模块放置在extras下

* 礼来礼往－－管理您的份子钱，目前第二好用的礼簿，最好用的是它弟弟 [蚂蚁礼簿](http://fir.im/zenw)
* 巨能贴－－安卓目前比较好用剪切板

## Demo

* [巨能贴](http://fir.im/akz8)
